# 小狗MBTI测试项目大纲

## 文件结构

### 主要HTML页面
1. **index.html** - 主页面
   - 欢迎界面和测试开始
   - 题目展示和答题功能
   - 结果计算和展示

2. **personalities.html** - 16种狗格总览页面
   - 所有狗格类型的网格展示
   - 详细的性格描述和匹配建议

### 资源文件
3. **resources/** - 资源文件夹
   - 狗狗相关插画和装饰图片
   - 背景纹理和图案

### 脚本文件
4. **main.js** - 主要JavaScript逻辑
   - 题目数据和计分系统
   - 页面切换和动画控制
   - 结果计算和展示逻辑

## 页面功能分解

### index.html 功能模块
1. **欢迎区域**
   - 标题和副标题
   - 开始测试按钮
   - 使用说明面板

2. **测试区域** (动态显示)
   - 题目进度条
   - 题目内容展示
   - A/B选项按钮
   - 上一题/下一题导航

3. **加载区域** (临时显示)
   - 加载动画
   - 进度指示

4. **结果区域** (动态显示)
   - 狗格类型标题
   - 狗狗形象展示
   - 四个维度得分
   - 性格特点描述
   - 匹配建议
   - 重新测试按钮

### personalities.html 功能模块
1. **导航栏**
   - 返回首页链接
   - 页面标题

2. **狗格网格**
   - 16种狗格卡片
   - 每个卡片包含：类型、名称、简短描述
   - 点击查看详情

3. **筛选和搜索**
   - 按维度筛选
   - 关键词搜索

### 计分系统
- 四个维度分别计分
- 每个维度4题
- A/B选择对应不同倾向
- 最终生成4字母组合

## 技术实现要点

### 页面状态管理
- currentPage: 当前页面状态
- currentQuestion: 当前题目索引
- answers: 用户答案数组
- scores: 各维度得分

### 动画效果
- 使用Anime.js实现页面切换
- CSS动画实现加载效果
- 按钮交互反馈

### 响应式设计
- 移动端优先设计
- 断点：768px (平板), 1024px (桌面)
- 弹性布局适配不同屏幕

### 数据持久化
- 使用localStorage保存测试结果
- 支持重新测试和结果回顾
